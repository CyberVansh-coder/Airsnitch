## 📥 Installation---->

## 📱 Termux (Android)
```
pkg update && pkg upgrade
pkg install git -y
pkg install bash -y
pkg install termux-api -y
git clone https://github.com/CyberVansh-coder/Airsnitch.git
cd AirSnitch
chmod +x airsnitch.sh
chmod +x modules/*.sh
bash airsnitch.sh
